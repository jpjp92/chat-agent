# 시각화 카드 전체화면 팝업 (Visualization Expand Modal)

## 기능 개요
시각화 카드(BioRenderer, ChemicalRenderer 등) 우하단에 expand 버튼을 추가하여
클릭 시 모달 팝업으로 해당 시각화를 전체화면 크기로 표시.

---

## 구현 방식

### 전반 구조
```
카드 우하단 expand 버튼 (fa-expand 아이콘)
    ↓ 클릭
App.tsx expandedViz state 업데이트 → { type, data }
    ↓
createPortal(document.body)로 VisualizationModal 마운트
    - backdrop blur + z-index 최상위
    - 동일 Renderer를 fullscreen 크기로 재렌더링
    - ESC 키 / X 버튼으로 닫기
```

### 상태 구조
```typescript
// App.tsx
const [expandedViz, setExpandedViz] = useState<{
    type: 'bio' | 'chemical' | 'physics' | 'constellation' | 'chart' | 'diagram' | 'drug';
    data: any;
} | null>(null);
```

### 주요 변경 파일
1. `App.tsx` — `expandedViz` state + `VisualizationModal` 컴포넌트 (Portal)
2. `components/ChatMessage.tsx` — 각 Renderer 래퍼에 expand 버튼 + `onExpand` 콜백
3. 각 Renderer 컴포넌트 — `isExpanded?: boolean` prop 수용 (크기 조정용)

### expand 버튼 위치
```tsx
// ChatMessage.tsx 내 각 Renderer 래퍼
<div className="relative group">
    <BioRenderer bioData={part.data} language={language} />
    <button
        onClick={() => onExpand?.({ type: 'bio', data: part.data })}
        className="absolute bottom-2 right-2 opacity-0 group-hover:opacity-100
                   bg-slate-700/80 text-white rounded-lg p-1.5 transition-opacity"
    >
        <i className="fa-solid fa-expand text-xs" />
    </button>
</div>
```

### VisualizationModal 구조 (Portal)
```tsx
// App.tsx — createPortal 사용
{expandedViz && createPortal(
    <div
        className="fixed inset-0 z-[9999] flex items-center justify-center p-4 sm:p-8"
        onClick={() => setExpandedViz(null)}
    >
        {/* Backdrop */}
        <div className="absolute inset-0 bg-black/60 backdrop-blur-md" />

        {/* Modal Content */}
        <div
            className="relative w-full max-w-5xl max-h-[90vh] overflow-auto
                       bg-white dark:bg-slate-900 rounded-3xl shadow-2xl
                       border border-slate-200 dark:border-white/10"
            onClick={(e) => e.stopPropagation()}
        >
            {/* Close Button */}
            <button
                onClick={() => setExpandedViz(null)}
                className="absolute top-4 right-4 z-10 w-8 h-8 rounded-full
                           bg-slate-100 dark:bg-white/10 flex items-center justify-center"
            >
                <i className="fa-solid fa-xmark text-sm" />
            </button>

            {/* Renderer (isExpanded=true) */}
            {renderExpandedContent(expandedViz)}
        </div>
    </div>,
    document.body
)}
```

## 렌더러별 고려사항

| 렌더러 | 팝업 적합성 | 특이사항 |
|--------|------------|---------|
| Bio 1D 서열 | ✅ 쉬움 | div 기반, 크기만 늘리면 됨 |
| Bio 3D PDB | ✅ 효과 큼 | NGL Stage가 stageRef에 마운트 → **새 인스턴스** 생성 필요 (기존 Stage와 독립) |
| Chemical (SMILES) | ✅ | canvas 기반, width/height prop 전달 |
| Physics | ⚠️ 주의 | Matter.js 엔진 두 개 동시 실행 → 모달 열 때 원본 pause 처리 권장 |
| Constellation | ✅ | canvas 크기 조정으로 처리 |
| Chart | ✅ | ApexCharts `width: '100%'` 설정 시 자동 채움 |
| Diagram | ✅ | canvas 2D, 크기 prop 전달 |
| Drug | ✅ | 정적 카드, 크기 확장만 |

### Physics pause 처리 방안
```typescript
// PhysicsRenderer.tsx
interface PhysicsRendererProps {
    // ...
    isPaused?: boolean; // 모달 팝업 시 원본 pause
}

useEffect(() => {
    if (isPaused) Matter.Runner.stop(runnerRef.current);
    else Matter.Runner.run(runnerRef.current, engineRef.current);
}, [isPaused]);
```
