# DEV_260421 — 성능 회귀 롤백 + 모델 통일 + 다국어 헤딩 재적용

## 배경

`cc04895` 이후 커밋들(9ece52f ~ 6a39b38)에서 발생한 회귀 현상:
- **날씨 응답 포맷 이상** — 이모지 규칙 변경 + CRITICAL 지침 추가로 모델 혼란
- **URL/PDF 요약 일부 잘림** — prompt.ts 포맷 구조 변경으로 요약 포맷 깨짐
- **PDF 분석 세션 타임아웃** — 복합적 회귀

## 수정 내용

### fix 1: prompt.ts 롤백 (cc04895 기준)
- `cc04895` 이후 누적된 prompt.ts 변경 전체 롤백
- 날씨 이모지 가이드, URL/PDF 요약 포맷이 원래 검증된 상태로 복원
- 유지된 비-prompt 개선사항: `geminiService.ts` 504 crash 방지, `useChatStream.ts` 재시도 딜레이 500ms, `generator.ts` maxOutputTokens 8192 복원·[N] 마커 제거·hasUrlContent 300자 threshold

### fix 2: 모델 gemini-2.5-flash 통일
- `api/_lib/agent/nodes/generator.ts`: `data_viz` 인텐트의 flash-lite 오버라이드 제거 → 모든 인텐트 flash 사용
- `api/_lib/agent/nodes/vision.ts`: `gemini-2.5-flash-lite` → `gemini-2.5-flash`
- `api/_lib/agent/prompt.ts`: 모델 자기소개 "Gemini 2.5 Flash-Lite" → "Gemini 2.5 Flash"
- 이유: flash-lite는 긴 문서·PDF 분석에서 품질 저하 및 세션 불안정 확인

### fix 3: URL 요약 헤딩 다국어 대응 재적용
- `getSystemInstruction` arrow 함수 → block 함수로 전환
- `URL_SUMMARY_LABELS` 맵 추가: Korean/English/Spanish/French 헤딩 정의
- URL_CONTENT 포맷 블록 헤딩을 `${lbl.summary}` / `${lbl.content}` / `${lbl.points}` 로 주입
- 기존 문제: `ENTIRE RESPONSE IN ENGLISH` 지시 시에도 한국어 헤딩("한 줄 요약") 고정 출력
- 설명 힌트(괄호 안 텍스트)는 LLM 전용이므로 한국어 유지 — 출력에 노출되지 않음
- 포맷 구조는 cc04895 원본 유지: `**헤딩**` 볼드 + `> (내용)` blockquote

## 변경 파일
- `api/_lib/agent/prompt.ts` — 롤백 + 다국어 헤딩 + 모델명
- `api/_lib/agent/nodes/generator.ts` — flash-lite 오버라이드 제거
- `api/_lib/agent/nodes/vision.ts` — 모델 통일
