# 세션 타이틀 모델 경량화 (2026-05-01 — 완료)

## 변경

### `api/summarize-title.ts` — SUMMARY_MODELS 순서 변경

```ts
// Before
const SUMMARY_MODELS = ['gemini-2.5-flash', 'gemini-2.5-flash-lite'];

// After
const SUMMARY_MODELS = ['gemini-2.5-flash-lite', 'gemini-2.5-flash'];
```

## 이유

- 타이틀 생성은 "5~15단어 명사형 제목" — 단순 요약 태스크
- `thinkingBudget: 0` 이미 적용 중이라 flash와 flash-lite 간 실질 품질 차이 없음
- lite가 더 빠르고 API quota 소모가 적음
- flash는 lite 실패 시 fallback으로 유지 (안전망 보존)

## 효과

- 타이틀 생성 레이턴시 소폭 감소
- API quota 효율 개선
