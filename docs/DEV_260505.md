# 개발 일지 (2026-05-05)

## 1. 프로필 이미지 업로드 용량 제한 상향 및 UI/UX 개선

### 원인 및 배경
- 사용자 프로필 설정에서 이미지 변경 시 파일 크기 제한이 2MB(`2 * 1024 * 1024`)로 다소 엄격하게 하드코딩되어 있었습니다.
- 스마트폰의 고화질 사진을 그대로 업로드하는 사용자 환경을 고려하여 용량 한도를 늘릴 필요성이 있었습니다.
- 또한, 허용 용량을 초과할 경우 우측 상단 토스트(Toast) 알림에 에러 텍스트 없이 느낌표 아이콘만 나타나는 버그가 존재했습니다.
- 마지막으로 공간이 제한적인 토스트 UI 특성상, 에러 메시지가 길면 말줄임표(...)로 뒤가 잘려 보이는 현상도 개선이 필요했습니다.

### 해결 및 변경 사항 (`components/Header.tsx`)
1. **용량 제한 10MB로 대폭 상향**
   - 파일 크기 검증 로직 변경: `2 * 1024 * 1024` → `10 * 1024 * 1024` (10MB).
   - 프로필 모달 내 이미지 규격 안내 문구를 4개 국어 모두 "최대 10MB" 등 해당 언어에 맞게 일괄 수정.
2. **누락된 다국어 에러 메시지(`sizeError`) 추가**
   - 기존 코드에서 `i18n` 딕셔너리에 `sizeError` 속성이 누락되어 토스트 메시지가 빈 값(`undefined`)으로 뜨던 문제를 해결했습니다.
3. **토스트 UI 맞춤형 간결한 메시지 작성**
   - 글자 잘림 현상을 방지하기 위해 4개 언어의 에러 메시지를 다음과 같이 매우 간결하게 최적화했습니다.
   - 🇰🇷 한국어: `"용량 초과 (최대 10MB)"`
   - 🇺🇸 영어: `"File too large (Max 10MB)"`
   - 🇪🇸 스페인어: `"Archivo muy grande (Máx 10MB)"`
   - 🇫🇷 프랑스어: `"Fichier trop lourd (Max 10Mo)"`

## 2. 프로필 모달 글래스모피즘(Glassmorphism) 스타일 적용
- **배경 개선**: 기존 단색 배경(`bg-white`, `bg-slate-900`)에서 반투명 배경 및 블러(`bg-white/80 dark:bg-slate-900/60 backdrop-blur-2xl`)로 변경하여 메인 헤더와 디자인 일관성 확보.
- **테두리 개선**: 단단한 느낌의 솔리드 보더를 반투명 화이트 테두리(`border-white/50 dark:border-white/10`)로 수정해 유리 패널의 질감 강화.
- **오버레이 강화**: 모달 등장 시 배경 블러를 `backdrop-blur-sm`에서 `backdrop-blur-md`로 한 단계 상향해 모달에 대한 시각적 집중도 향상.
- **버튼 및 입력창 레이어링**: 텍스트 인풋과 토글 버튼 등의 불투명한 회색 배경들을 투명도가 있는 색상(`bg-black/5 dark:bg-white/5`)으로 교체하여 깊이감 있고 부드러운 인터랙션 구현.
