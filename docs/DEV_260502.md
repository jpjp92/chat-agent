## 2. PhysicsRenderer 버그 수정 (`components/PhysicsRenderer.tsx`)

### 버그 1 — `ceiling` 바디 월드 미추가 (수정 완료)

```ts
// Before: ceiling 생성만 되고 Composite.add에 누락
Matter.Composite.add(engine.world, [...bodies, ground, leftWall, rightWall, mouseConstraint]);

// After: ceiling 포함
Matter.Composite.add(engine.world, [...bodies, ground, leftWall, rightWall, ceiling, mouseConstraint]);
```

**증상**: 위로 발사된 물체가 천장 경계 없이 화면 밖으로 사라짐.

### 버그 2 — 사각형 오브젝트 라벨 수직 위치 오류 (수정 완료)

```ts
// Before: rectangle도 radius || 20 사용 → 큰 사각형은 라벨이 물체 안에 겹침
context.fillText(obj.label, x, y - (obj.radius || 20) * scale - staggerOffset);

// After: 타입별 분기
const halfSize = obj.type === 'circle'
    ? (obj.radius || 20)
    : (obj.height || 40) / 2;
context.fillText(obj.label, x, y - halfSize * scale - staggerOffset);
```

### 성능 — `isInclinedPlane` O(n²) 계산 → O(n) (수정 완료)

```ts
// Before: objects.map() 내부에서 매 오브젝트마다 전체 스캔 (n² 탐색)
const bodies = physicsData.objects.map(obj => {
    const isInclinedPlane = physicsData.objects.some(...); // 매번 재탐색
    ...
});

// After: map 외부에서 1회만 계산
const isInclinedPlane = physicsData.objects.some(...);
const bodies = physicsData.objects.map(obj => { ... });
```

### 잔여 디버그 `console.log` 3건 제거

경사면 템플릿 감지 시 출력되던 이모지 로그(`🔧`, `📐`, `📦`) 제거.

## 3. 세션 초기화 "연결에 실패했습니다" 에러 화면 수정

### 원인 분석

**경로 1 — `initAuth().catch()` 핸들러** (`useAuthSession.ts`)

`initAuth()` 내부에서 예외 발생 시 catch 블록이 `setIsAuthLoading(false)`만 호출하고 `setCurrentUser`를 명시 호출하지 않아, `isAuthLoading=false + currentUser=null` 조합으로 에러 화면 표시.

**경로 2 — localStorage 파싱 실패 후 `else` 미진입** (`useAuthSession.ts`)

`JSON.parse(savedUser)` 실패 시 `catch` 블록에서 localStorage만 제거하고 신규 게스트 생성(`else` 분기)으로 진입하지 않아 `user=null`인 채로 진행.

**경로 3 — `handleReset` React 상태 선행 업데이트** (`App.tsx`) ← 실제 재현 경로

```ts
// Before: clearStoredUser()가 setCurrentUser(null) 호출 → reload 전에 리렌더 발생
const handleReset = () => {
    clearStoredUser();           // → setCurrentUser(null), isAuthLoading=false 그대로
    window.location.reload();    // 리렌더 후 실행
};
// 결과: isAuthLoading=false + currentUser=null → 에러 화면 0.1초 순간 표시
```

### 수정 내용

**`App.tsx` — `handleReset` React 상태 업데이트 제거**

```ts
// After: React 상태 변경 없이 localStorage만 직접 제거 후 reload
const handleReset = () => {
    localStorage.removeItem('gemini_chat_user');
    localStorage.removeItem('chat_sessions_cache_v1');
    window.location.reload();
};
```

**`src/hooks/useAuthSession.ts` — localStorage 파싱 실패 시 게스트 재생성 폴백**

- `createGuestUser()` 헬퍼 분리
- `JSON.parse` 실패 시 `createGuestUser()` 호출로 폴백 (기존: 그냥 null 반환)
- `.catch()` 핸들러에 `setCurrentUser(null)` 명시 추가

**`components/LoadingScreen.tsx` — `children` prop 추가**

에러 화면에 "다시 시도" 버튼 렌더링 지원. 에러 상태일 때 애니메이션 점(bounce) 대신 버튼 표시.
